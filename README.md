
# greetings

<!-- badges: start -->
<!-- badges: end -->

The goal of `greetings` is to say aloha to friend.

## Installation

You can install this package with:

``` r
remotes::install_github("brunj7/greetings")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(greetings)
say_aloha("Allison")
#> Aloha, Allison 🌴 ☀️ 🌊
```
