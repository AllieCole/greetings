
# aloahjb

<!-- badges: start -->
<!-- badges: end -->

The goal of aloahjb is to say aloha to friend.

## Installation

You can install this package with:

``` r
remotes::install_github("brunj7/aloahjb")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(aloahjb)
say_aloha("Allison")
```
